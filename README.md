# Tichi QA Intern Technical Assignment

## About This Project

This repository contains my submission for the **QA Intern Technical Assignment**.

In this assignment, I tested the Login and Signup functionality of the Tichi web application. I created manual test cases, reported a bug, and automated the Login functionality using Playwright with JavaScript.

**Application URL:**  
https://tichi-app-webapp-stage.web.app/

---

## What I Completed

### Manual Testing

- Created test cases for Login and Signup pages.
- Covered both positive and negative test scenarios.
- Verified input validations and error messages.

### Bug Reporting

Reported a bug where the application accepts an invalid email format during login.

The bug report includes:
- Bug Summary
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Priority

### Automation Testing

Automated the Login functionality using **Playwright + JavaScript**.

The automation covers:
- Valid Login
- Invalid Login
- Empty Email
- Empty Password
- Invalid Email Format Validation

## How to Run the Project

### Clone the repository

```bash
git clone https://github.com/praveenpraveen35352-dot/Tichi_App_Qa_Assignment.git
```

### Go to the project folder

```bash
cd Tichi_App_Qa_Assignment
```

### Install dependencies

```bash
npm install
```

### Install Playwright

```bash
npx playwright install
```

### Run the tests

```bash
npx playwright test
```

### View the HTML Report

```bash
npx playwright show-report
```
